#### 工具

使用head插件或都kibana都可以,head更适合直观的查看信息,kibana更适合操作命令


#### 创建索引、插入数据

--查看有哪些索引
````
GET /_cat/indices?v 
````
--创建
````
PUT /book?pretty
````

--删除
````
DELETE /book?pretty

````

--创建结构化类型type
````
POST book/novel/_mappings
{
  "novel": {
    "properties": {
      "title": {
        "type": "text"
      }
    }
  }
}
````

--完整的创建

````
PUT /book
{
  "settings":{
    "number_of_shards":3,
    "number_of_replicas":1
  },
  "mappings":{
    "novel":{
      "properties":{
        "word_count":{
          "type":"integer"
        },
        "author":{
          "type":"keyword"
        },
        "title":{
          "type": "text"
        },
        "publish_date":{
          "type": "date",
          "format":"yyyy-MM-dd HH:mm:ss||yyyy-MM-dd||epoch_millis"
        }
      }
    }
  }
}

````
--nove中添加一条数据(doc)
````
PUT /book/novel/1
{
  "word_count":160090,
  "author":"alex",
  "title":"九阴真经",
  "publish_date":"1500-05-6"
  
}
````

#### 修改



````
POST /book/1/_update
{
  "doc":{
    "author":"jackzhang1"
  }
}
````

也可直接使用put进行替换

````
PUT /book/novel/1
{
  "word_count":160090,
  "author":"alex",
  "title":"并发编程1",
  "publish_date":"2000-05-6"
  
}
````

#### 删除

````
DELETE /book/novel/1

````
